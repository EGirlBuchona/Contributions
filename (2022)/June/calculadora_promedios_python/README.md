# Average Calculator in Python

This Python calculator allows the user to calculate the average of entered grades or any set of numeric values. It is a console-based application.

## Instructions

1. Run the `calculadora_promedios.py` script.
   ```bash
   python calculadora_promedios.py
   ```

2. Enter grades or values one by one. Type 's' to finish and get the average.

## Features

- **Average Calculation:** Calculates the average of entered values.
- **Error Handling:** Provides feedback for invalid inputs.

## Dependencies

- Python 3.6 or higher

# Calculadora de Promedios en Python

Esta calculadora en Python permite al usuario calcular el promedio de calificaciones ingresadas o cualquier conjunto de valores numéricos. Es una aplicación de consola.

## Instrucciones

1. Ejecute el script `calculadora_promedios.py`.
   ```bash
   python calculadora_promedios.py
   ```

2. Ingrese las calificaciones o valores uno por uno. Escriba 's' para terminar y obtener el promedio.

## Funcionalidades

- **Cálculo de Promedio:** Calcula el promedio de los valores ingresados.
- **Manejo de Errores:** Proporciona retroalimentación para entradas no válidas.

## Dependencias

- Python 3.6 o superior
