# Text Summary Script in Python

This Python script reads a text file and generates a summary, including statistics such as word count, sentence count, paragraph count, and the most common words.

## Instructions

1. Run the `resumen_texto.py` script.
   ```bash
   python resumen_texto.py
   ```

2. Enter the path of the text file when prompted.
3. The script will process the file and display the summary.

## Features

- **Text Analysis:** Counts words, sentences, and paragraphs in a text file.
- **Word Frequency:** Identifies the five most common words in the text.
- **Error Handling:** Provides feedback for invalid file paths.

## Dependencies

- Python 3.6 or higher

# Script para Generar Resúmenes de Texto en Python

Este script en Python lee un archivo de texto y genera un resumen, incluyendo estadísticas como el número de palabras, oraciones, párrafos y las palabras más comunes.

## Instrucciones

1. Ejecute el script `resumen_texto.py`.
   ```bash
   python resumen_texto.py
   ```

2. Ingrese la ruta del archivo de texto cuando se le solicite.
3. El script procesará el archivo y mostrará el resumen.

## Funcionalidades

- **Análisis de Texto:** Cuenta palabras, oraciones y párrafos en un archivo de texto.
- **Frecuencia de Palabras:** Identifica las cinco palabras más comunes en el texto.
- **Manejo de Errores:** Proporciona retroalimentación para rutas de archivo no válidas.

## Dependencias

- Python 3.6 o superior
