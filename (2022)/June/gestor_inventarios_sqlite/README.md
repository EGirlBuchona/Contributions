# Inventory Manager with SQLite

This Python project manages inventory using an SQLite database. It includes full CRUD functionality to manage products, quantities, and prices.

## Instructions

1. Run the `gestor_inventarios.py` script.
   ```bash
   python gestor_inventarios.py
   ```

2. Use the menu options to:
   - Add products
   - List all products
   - Update product information
   - Delete products

## Features

- **Database Integration:** Uses SQLite for persistent storage.
- **CRUD Operations:** Full support for create, read, update, and delete functionalities.
- **Interactive Menu:** User-friendly interface for managing inventory.

## Dependencies

- Python 3.6 or higher
- SQLite (included with Python standard library)

# Gestor de Inventarios con SQLite

Este proyecto en Python gestiona inventarios utilizando una base de datos SQLite. Incluye funcionalidad CRUD completa para manejar productos, cantidades y precios.

## Instrucciones

1. Ejecute el script `gestor_inventarios.py`.
   ```bash
   python gestor_inventarios.py
   ```

2. Use las opciones del menú para:
   - Agregar productos
   - Listar todos los productos
   - Actualizar información de productos
   - Eliminar productos

## Funcionalidades

- **Integración con Base de Datos:** Utiliza SQLite para almacenamiento persistente.
- **Operaciones CRUD:** Soporte completo para crear, leer, actualizar y eliminar datos.
- **Menú Interactivo:** Interfaz fácil de usar para gestionar inventarios.

## Dependencias

- Python 3.6 o superior
- SQLite (incluido en la biblioteca estándar de Python)
